# DevSecOps CI/CD Pipeline

Projet PFA — Ai Big Solution  — Mimouh Aroua & Zraidi Salma

## Modules
- Module A : Threat Modeling (OWASP Threat Dragon + STRIDE)
- Module B : CI/CD Pipeline automatisé (GitHub Actions)
- Module C : Exploitation (Burp Suite)
- Module D : Centralisation (DefectDojo)
- Module E : Adaptation Big Solutions
- Module F : Security by Design Guide

## Stack
- Pipeline : GitHub Actions
- Scanners : Semgrep, Trivy, Gitleaks, OWASP ZAP
- Gestion : DefectDojo + PostgreSQL + Redis
- Dashboard : Flask (backend) + React (frontend)
